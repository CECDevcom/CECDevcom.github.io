How to Setup Github Pages with NameCheap

So this blog is hosted on Github Pages and everything is working well and good. The one thing that you would notice diffrent from any new fresh github pages setup is the domain name on the top.


Yep that is one Custom Domain curtesy of NameCheap (free with Student Developer Pack), and let's set it up.



Setup a CNAME Record


Create a New file CNAME (Mind the Capitals) and save it to the root of the repo, with the following content.


yourawesomedomain.me
www.yourawesomedomain.me

Don't forget to commit it to the Repository.
